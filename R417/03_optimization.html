例題3-1：¶

勾配法を実行する関数optimizationを作成せよ。
ただし、引数と戻り値は以下の通りとする。

[引数]
f: 目的関数
df: 目的関数の1次導関数
x0: 初期点
alpha: 学習率
max_itr: 更新回数

[戻り値]
x_seq: 最適化の過程を記録した変数xの点列

In [9]:
import pandas as pd
import numpy as np

def optimization(f, df, x0, alpha, max_itr):
    x_seq = [x0]
    for i in range(max_itr):
        x_seq.append(x_seq[i] - alpha * df(x_seq[i]))
    return x_seq
